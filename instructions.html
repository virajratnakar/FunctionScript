🚀 Learn the Function Language

This language lets you create UI, apps, and games using simple function-like commands.

📌 Basic Idea

Each line is a function:

functionName(argument1, argument2)

📝 Text & Display

text(Hello world)
title(My App)
subtitle(Small heading)
bold(Important)
italic(Nice text)

🔘 Buttons

button(Click me)
button(Open Google, open(https://google.com))

🔎 Search & Links

search(cats)
open(https://youtube.com)

📋 Lists

list(tasks)
addTask(Do homework)
addTask(Learn coding)

🖼 Images & UI

img(https://via.placeholder.com/150)
color(red, Hello!)
box(100px,100px,blue)
card(Title, Description)

🎮 Game System

enable(game)
startGame()
attack()
heal(10)

🎲 Fun Commands

emoji()
rainbow(Hello)
typewriter(Loading...)
random(1, 100)

⚡ Tips

  • Each line runs separately
  • You can combine commands with buttons
  • Try experimenting!

🔥 Example


title(My First App)
text(Welcome!)
button(Click me, text(You clicked!))
